# rust-samples

List of samples for rust learning

## Installations

[Rust/Cargo](https://www.rust-lang.org/ "Rust/Cargo")

## Commands

Create new project:

    cargo new hello_world --bin

Build project where Cargo.toml is:

    cargo build

Run project(implicit build):

    cargo run
